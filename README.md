# myVimEditor
Exportable Vim Editor (Autocompletion, Syntax Highlighting, Linter, Directory...)

## Installation

**1.** Run Bash Script

If Linux AMI / Redhat:

``` console
bash makeVimAWS.sh
```
If Debian / Ubuntu

``` console
bash makeVimPi.sh
```
**2.** Open Vim and type ***:PluginInstall***

**Important: Run compileVim.sh if Vim version on OS doesn't support python 2/3**
